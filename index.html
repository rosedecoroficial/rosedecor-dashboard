⚙️ Integrações & Configuração
Como conectar cada fonte de dados ao dashboard
🔗 Conectar sua planilha ao dashboard
1
Cole a URL do seu Apps Script aqui
Após configurar o Apps Script na sua planilha (veja abaixo), você recebe uma URL pública. Cole ela aqui e clique Conectar. O dashboard passa a ler seus dados reais automaticamente.
📊 Passo 1 — Estrutura da planilha
Sua planilha precisa ter 4 abas com esses nomes exatos. Cada fonte de dados alimenta uma aba diferente.

🤖 Passo 2 — Apps Script (a ponte)
a
Na planilha: Extensões → Apps Script
Abre o editor de código. Apague tudo que estiver lá e cole o código abaixo.
function doGet(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const abas = ['leads', 'meta_ads', 'vendas', 'config'];
  const out = {};
  abas.forEach(n => {
    const sh = ss.getSheetByName(n);
    if(!sh) return;
    const d = sh.getDataRange().getValues();
    const h = d[0];
    out[n] = d.slice(1).filter(r=>r[0]).map(r=>{
      const o={}; h.forEach((k,i)=>o[k]=r[i]); return o;
    });
  });
  return ContentService
    .createTextOutput(JSON.stringify(out))
    .setMimeType(ContentService.MimeType.JSON);
}
b
Implantar → Nova implantação → Aplicativo da Web
Execute como: Eu · Quem tem acesso: Qualquer pessoa · Clique Implantar · Autorize quando pedir.
Copie a URL gerada — ela termina em /exec — e cole no campo lá em cima.
📊 Passo 3 — Adveronix (Meta Ads → Sheets)
1
Instalar o complemento Adveronix
Na planilha: Extensões → Complementos → Instalar complementos → buscar "Adveronix" → Instalar.
É gratuito. Não precisa de cartão.
2
Extensões → Adveronix → Launch → conectar Facebook Ads
Clique em Connect → Facebook Ads → autorize o acesso à sua conta de anúncios.
3
Criar relatório com as métricas certas
Destination: aba meta_ads · Breakdown: Day + Campaign + Ad · Métricas:
datecampaign_nameadset_namead_namespendimpressionsclicksctrcpmreachfrequencyleads
4
Configurar agendamento automático
No Adveronix, clique em Schedule → Daily → 8:00 AM. A partir daí ele atualiza sozinho todo dia sem você fazer nada.
O Meta libera dados com ~3h de atraso. Por isso roda às 8h — traz os dados completos de ontem.
💳 Passo 4 — Make.com + Hubla (Vendas → Sheets)
1
Criar conta gratuita no Make.com
make.com → Sign up grátis. Plano free tem 1.000 operações/mês — suficiente para vários lançamentos.
2
Criar cenário: Hubla → Google Sheets
No Make: Create scenario → Webhooks → Custom webhook. Copie a URL do webhook gerada.
3
Configurar webhook na Hubla
Na Hubla: Configurações → Integrações → Webhooks → Adicionar. Cole a URL do Make. Evento: purchase.approved.
A partir de agora, cada venda aprovada na Hubla dispara automaticamente o Make.
4
Adicionar ação Google Sheets no Make
Após o webhook, adicione: Google Sheets → Add a Row → selecione sua planilha → aba vendas → mapeie: date, name, email, product, value, status, payment.
Venda aprovada → aparece no dashboard em menos de 60 segundos.
